# ADR 0003: Apache Airflow вместо Prefect

## Статус
Принято.

## Решение
Выбран Apache Airflow 2.9.

## Обоснование
- Больше документации и туториалов на русском
- TaskFlow API удобен для Python-тасков
- Широко используется в индустрии
- Бесплатный self-hosted вариант

## Альтернативы
- Prefect: проще в настройке, но меньше сообщество
